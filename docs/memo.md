## GA4データ取得手順
1. Google Analytics (GA4) にログインし、対象のアカウント（ritotabi）とプロパティ（ritotabi）を選択します。
2. 左側のメニューから 「レポート」 アイコン（グラフのマーク）をクリックします。
3. 「エンゲージメント」 > 「ページとスクリーン」 を選択します。
4. 右上の日付選択ツールで、取得したい 期間（例：2026/03/01 - 2026/03/31） を設定し、「適用」をクリックします。
5. グラフの下にあるテーブルの左上（ディメンション選択）が 「ページパスとスクリーン クラス」 になっていることを確認します。
   - デフォルトが「ページタイトルとスクリーン クラス」になっている場合は、プルダウンから変更してください。
6. レポートの右上にある 「共有」アイコン（上向きの矢印のようなマーク）をクリックします。
7. 表示されたメニューから 「ファイルをダウンロード」 を選択し、「CSV をダウンロード」 をクリックします。

## Google Search Console データ取得手順
1. Google Search Console にログインし、対象のプロパティ（ritotabi）を選択します。
2. 左側のメニューから 「パフォーマンス」>「検索結果」 を選択します。
3. 左上の合計クリック数、合計表示回数、平均CTR、平均掲載順位をクリックして有効化します。
4. 左上の日付選択ツールで、取得したい 期間（例：2026/03/01 - 2026/03/31） を設定し、「適用」をクリックします。
5. グラフの下にあるテーブルの左上（ディメンション選択）が 「ページ」 になっていることを確認します。
   - デフォルトが「クエリ」になっている場合は、プルダウンから変更してください。
6. レポートの右上にある 「共有」アイコン（上向きの矢印のようなマーク）をクリックします。
7. 表示されたメニューから 「ファイルをダウンロード」 を選択し、「CSV をダウンロード」 をクリックします。

## Bing Webmaster Tools データ取得手順
1. Bing Webmaster Tools にログインし、対象のプロパティ（ritotabi）を選択します。
2. 左側のメニューから 「Search Performance」 を選択します。
3. ページ下部のList Byで「Pages」を選択します。
4. 左上の日付選択ツールで、取得したい 期間（例：2026/03/01 - 2026/03/31） を設定し、「適用」をクリックします。
5. 画面右上にある「Export」ボタンをクリックし、「CSV」を選択します。
