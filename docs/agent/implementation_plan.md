# 評価日の表示追加

品質評価のトップページ（QualityTab）の各カードに、評価実施日（`evaluatedAt`）を表示するように変更します。

## ユーザーレビューが必要な項目
- 特になし（既存のレイアウトに馴染む形で追加します）

## 変更内容

### UIコンポーネント

#### [MODIFY] [QualityTab.tsx](file:///home/mune1/dev/ritotabi/eval_site/ritotabi_analysis/src/components/QualityTab.tsx)
- グリッド表示の各カード内にあるメタ情報エリア（言語やタイプが表示されている箇所）に、評価日を追加します。
- 既存の `QualityDetail` コンポーネントでの表示形式（`評価：2026-04-07` 等）に合わせます。

## 検証計画

### 手動確認
- `npm run dev` でローカルサーバーを起動し、ブラウザで品質評価タブを開く。
- 各カードに「評価：YYYY-MM-DD」といった形式で日付が表示されていることを確認する。
- レイアウトが崩れていない（折り返しなどが適切か）を確認する。
