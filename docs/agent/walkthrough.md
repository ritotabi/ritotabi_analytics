# PV予想値の再キャリブレーション完了報告

統計データおよび市場の競合性を再評価し、シミュレーターの基盤となるPV予測値を現実に即した形に修正しました。

## 実施内容

### 1. ベースラインPVの抜本的修正 (`baseline-pv.ts`)
入域観光客数の統計（石垣島140万、宮古島120万、ホイアン440万、コンダオ60万）をベースに、各ストリームのポテンシャルを再計算しました。

- **石垣島・宮古島 (JP) の底上げ**: 
  - 国内主要離島としての検索ボリュームを反映し、ベースラインを従来の約4倍に引き上げました。
- **英語ベースラインの適正化**: 
  - 英語圏の検索ボリュームを「日本語の約1/3」を上限の目安として設定。
  - ホイアン(EN)をグローバルな大拠点（10,000 PV/月）として再定義。
- **ニッチ市場（コンダオ島等）の下方修正**: 
  - 日本人客が極めて少ない実態に合わせ、ベースラインを 1/3 程度に抑制。

### 2. 個別評価ファイルの見直し (`src/evaluations/*.json`)
ベースラインを足し合わせた「合計PV」が、実際の来島者数と比較して乖離しすぎないよう、英語ページの成長予測（Evaluation PV）をトーンダウンしました。

- **石垣島(EN)**: 通常予測(pn)の最大値を 28,000 → **12,000** へ修正。
- **コンダオ島(EN)**: 通常予測(pn)の最大値を 7,000 → **2,000** へ修正。

## 修正後の市場規模階層（2028年3月・合計予測PV目安）

| 順位 | ストリーム | 合計予測PV (推計値) | 状況 |
| :--- | :--- | :--- | :--- |
| **1位** | **石垣島 (JP)** | **約70,000** | 国内最大の主戦場として圧倒的なシェア |
| **2位** | **宮古島 (JP)** | **約60,000** | 石垣島に並ぶ国内強豪拠点 |
| **3位** | **ホイアン (EN)** | **約37,000** | グローバルな観光ハブとしての高いリーチ |
| **4位** | **石垣島 (EN)** | **約25,000** | インバウンド成長の主要チャネル |
| **5位** | **コンダオ (EN)** | **約4,500** | 世界的な隠れ家離島としてのニッチ独占 |
| **6位** | **コンダオ (JP)** | **約1,200** | 唯一無二の日本語情報源としてのニッチシェア |

## 検証結果
修正後のシミュレーションでは、**「日本国内向けの石垣・宮古が最大、次いでインバウンドのホイアン、ニッチなコンダオ」**という、実来訪者数とRITOTABIの戦略（専門性によるシェア獲得）が矛盾なく共存するデータ構造になっています。

> [!IMPORTANT]
> この修正により、これまで「英語ページが多すぎる」「石垣とコンダオの日本語が変わらない」と感じられていた乖離が解消され、より説得力のある事業シミュレーションが可能になりました。
